4.运行结果

Adjusted Rand Index, ARI 效果
ARI = 1 聚类结果与真实分类完全一致
ARI > 0.9 通常认为聚类结果非常好
0.7 ≤ ARI ≤ 0.9 表示聚类结果较好
0.5 ≤ ARI < 0.7 表示聚类结果一般，可能需要改进
0 < ARI < 0.5 表示聚类结果较差，但优于随机分配
ARI ≤ 0 表示聚类结果无效或与真实标签不一致

5.复盘与分析

用K-means++的方法对质心的初始化进行了改进，对时间影响不大并且在部分数据集中提高了模型的性能，然而好几个算法中调整兰德指数(ARI)还是很低，可能是k-means算法本身的局限性。
