ARI = 1 聚类结果与真实分类完全一致
ARI > 0.9 通常认为聚类结果非常好
0.7 ≤ ARI ≤ 0.9 表示聚类结果较好
0.5 ≤ ARI < 0.7 表示聚类结果一般，可能需要改进
0 < ARI < 0.5 表示聚类结果较差，但优于随机分配
ARI ≤ 0 表示聚类结果无效或与真实标签不一致
Davies-Bouldin Index, DBI 效果
DBI < 1 通常认为聚类效果非常好
1 ≤ DBI ≤ 2 聚类结果尚可
DBI > 2 通常表示聚类质量较差，可能需要改进

5.复盘与分析

不同的模型受到 ϵ\epsilonϵ 影响较大，我的目的只是单纯理解模型架构，所以没有对每个数据集进行针对性调参，使得模型的可靠性都不高。DBSCAN算法 还有一个问题就是先标记的簇的元素数量会远多于后标记的簇，因为同为某簇边界点的元素会先被簇标签较小的簇标记。要改善这一问题，可以通过 HDBSCAN（Hierarchical DBSCAN）算法 来弥补这一缺陷。
